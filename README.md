# 📦 Analisis E-Commerce Olist Brazil

## Deskripsi
Dashboard analisis data transaksi e-commerce Olist Brazil
mencakup tren penjualan, kepuasan pelanggan, RFM Analysis,
dan Geospatial Analysis periode 2017-2018.

## Setup Environment

### 1. Buat Virtual Environment
python -m venv venv
### 2. Aktifkan Virtual Environment
Windows: venv\Scripts\activate
Mac/Linux: source venv/bin/activate
### 3. Install Dependencies
pip install -r requirements.txt
## Run Dashboard
cd dashboard
streamlit run dashboard.py
