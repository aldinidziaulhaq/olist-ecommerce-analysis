# Analisis E-Commerce Olist Brazil

## Deskripsi
Dashboard analisis data transaksi e-commerce Olist Brazil
mencakup tren penjualan, kepuasan pelanggan, RFM Analysis,
dan Geospatial Analysis periode 2017-2018.

## Setup Environment
1. Install library yang dibutuhkan:
pip install -r requirements.txt

2. Jalankan dashboard:
cd dashboard
streamlit run dashboard.py
